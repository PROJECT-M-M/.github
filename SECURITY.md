# Politique de sécurité

## Signaler une vulnérabilité

Si tu trouves une faille de sécurité dans un de nos repos, **n'ouvre pas d'issue publique**.

À la place :
1. Va sur l'onglet **Security** du repo concerné
2. Clique sur **"Report a vulnerability"** (Private Vulnerability Reporting)
3. Décris la faille avec autant de contexte que possible

On s'engage à :
- Accuser réception sous 72h
- Évaluer la sévérité
- Fixer et publier un patch le plus vite possible (sous 30 jours pour les CVE critiques)

## Versions supportées

On supporte uniquement la branche `main` de chaque projet.
